# Terminal Tools & Configurations

A collection of free and open-source terminal tools, shell configurations, and CLI utilities to boost your productivity as a developer.

## Shell Frameworks & Prompts

- [Oh-my-Zsh](https://github.com/ohmyzsh/ohmyzsh) - Framework for managing your Zsh configuration with plugins and themes
- [Oh-my-Zsh Themes](https://github.com/ohmyzsh/ohmyzsh/wiki/Themes) - Collection of themes for Oh-my-Zsh
- [Powerlevel10k](https://github.com/romkatv/powerlevel10k) - Fast and flexible Zsh theme with instant prompt
- [Oh-my-Bash](https://github.com/ohmybash/oh-my-bash) - Framework for managing your Bash configuration (similar to Oh-my-Zsh)
- [Fish Shell](https://github.com/fish-shell/fish-shell) - Smart and user-friendly command line shell with autosuggestions
- [Oh-my-Fish](https://github.com/oh-my-fish/oh-my-fish) - Plugin framework for Fish shell
- [Starship](https://github.com/starship/starship) - Minimal, blazing-fast, cross-shell prompt written in Rust
- [Zsh Autosuggestions](https://github.com/zsh-users/zsh-autosuggestions) - Fish-like autosuggestions for Zsh
- [Zsh Syntax Highlighting](https://github.com/zsh-users/zsh-syntax-highlighting) - Syntax highlighting for the Zsh shell

## Terminal Emulators

- [iTerm2](https://iterm2.com/) - macOS terminal replacement with GPU rendering, split panes, and search (macOS)
- [Alacritty](https://github.com/alacritty/alacritty) - GPU-accelerated terminal emulator, cross-platform
- [Kitty](https://github.com/kovidgoyal/kitty) - Fast, feature-rich, GPU-based terminal emulator
- [Warp](https://www.warp.dev/) - Modern terminal with AI command search and IDE-like editing (macOS/Linux)
- [Windows Terminal](https://github.com/microsoft/terminal) - Modern terminal for Windows with tabs, themes, and GPU rendering
- [Hyper](https://github.com/vercel/hyper) - Electron-based terminal built on web technologies

## Terminal Multiplexers

- [Tmux](https://github.com/tmux/tmux) - Terminal multiplexer for managing multiple sessions, windows, and panes
- [Tmux Plugin Manager (TPM)](https://github.com/tmux-plugins/tpm) - Plugin manager for Tmux
- [Oh-my-Tmux](https://github.com/gpakosz/.tmux) - Self-contained, pretty, and versatile Tmux configuration
- [Zellij](https://github.com/zellij-org/zellij) - Modern terminal multiplexer written in Rust with a layout system
- [GNU Screen](https://www.gnu.org/software/screen/) - Classic terminal multiplexer

## File Navigation & Search

- [fzf](https://github.com/junegunn/fzf) - General-purpose command-line fuzzy finder
- [ripgrep (rg)](https://github.com/BurntSushi/ripgrep) - Extremely fast text search tool (grep replacement)
- [fd](https://github.com/sharkdp/fd) - Simple, fast, and user-friendly alternative to `find`
- [zoxide](https://github.com/ajeetdsouza/zoxide) - Smarter `cd` command that learns your most used directories
- [ranger](https://github.com/ranger/ranger) - Console file manager with Vi keybindings
- [nnn](https://github.com/jarun/nnn) - Blazing-fast terminal file manager
- [broot](https://github.com/Canop/broot) - Interactive tree view for navigating directories

## Better Defaults (Modern CLI Replacements)

- [bat](https://github.com/sharkdp/bat) - A `cat` clone with syntax highlighting and Git integration
- [eza](https://github.com/eza-community/eza) - Modern replacement for `ls` with colors, icons, and Git status
- [delta](https://github.com/dandavison/delta) - Syntax-highlighting pager for `git diff` output
- [dust](https://github.com/bootandy/dust) - More intuitive version of `du` (disk usage)
- [duf](https://github.com/muesli/duf) - Better `df` alternative for disk usage/free
- [htop](https://github.com/htop-dev/htop) - Interactive process viewer (better `top`)
- [btop](https://github.com/aristocratos/btop) - Resource monitor with a rich UI
- [tldr](https://github.com/tldr-pages/tldr) - Simplified and community-driven man pages
- [httpie](https://github.com/httpie/cli) - Human-friendly HTTP client (better `curl` for APIs)
- [jq](https://github.com/jqlang/jq) - Lightweight command-line JSON processor
- [procs](https://github.com/dalance/procs) - Modern replacement for `ps`

## Git CLI Tools

- [lazygit](https://github.com/jesseduffield/lazygit) - Simple terminal UI for Git commands
- [tig](https://github.com/jonas/tig) - Text-mode interface for Git
- [gh](https://github.com/cli/cli) - GitHub's official command-line tool
- [git-extras](https://github.com/tj/git-extras) - Useful extra Git commands (summary, changelog, etc.)
- [diff-so-fancy](https://github.com/so-fancy/diff-so-fancy) - Better-looking diffs for Git

- [NvChad](https://github.com/NvChad/NvChad) - Blazing fast Neovim config providing solid defaults and a beautiful UI

## Productivity & Utilities

- [direnv](https://github.com/direnv/direnv) - Load/unload environment variables based on directory
- [asdf](https://github.com/asdf-vm/asdf) - Manage multiple runtime versions with a single CLI tool
- [ncdu](https://dev.yorhel.nl/ncdu) - NCurses disk usage analyzer
- [figlet](https://github.com/cmatsuoka/figlet) - Generate text banners in the terminal
