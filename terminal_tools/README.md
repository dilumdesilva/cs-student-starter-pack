# Useful Environment Configs & Terminal Tools

This collection contains useful configurations, terminal tools, and CLI utilities that you can use on your laptop/pc, virtual, container or cloud environment to improve your productivity as a developer.

## How to contribute to the collection.
If you are already using any configuration on your IDE or terminal or any other environment which has helped you to improve your productivity of day to day work as a developer or a software engineer. Feel free to share your configs with others by contributing to this section.

In order to contribute please follow the steps mentioned below.
- create a md/markdown file within the terminal_tools folder relevant to the config type (if a file is already existing for that config type please update that file).
- update below list and link your newly created file.
- send us a pull request for review (update the relevant issue if there's an issue has created to track contributions to this section).

1. [Terminal Tools & Configurations](terminal_configs.md) - Shell frameworks, terminal emulators, multiplexers, modern CLI replacements, Git tools, editors, and more
2. IDE Configs

   - [Visual Studio Code]()
