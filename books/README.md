# Books

A collection of free and open-source books for computer science and software engineering students.

## Free Book Collections

- [Free Programming Books](https://github.com/EbookFoundation/free-programming-books) - Freely available programming books in many languages
- [GoBooks](https://github.com/dariubs/GoBooks) - List of Go programming books
- [Free Security eBooks](https://github.com/Hack-with-Github/Free-Security-eBooks) - Free security and hacking eBooks

## Computer Science Fundamentals

- [Open Source CS Degree](https://github.com/ForrestKnight/open-source-cs) - Free courses from top universities for a complete CS curriculum
- [Teach Yourself Computer Science](https://teachyourselfcs.com/) - Guide to self-studying computer science
- [Computer Science from the Bottom Up](https://www.bottomupcs.com/) - Understanding computers from low-level to high-level

## Software Engineering & Design

- [Clean Code JavaScript](https://github.com/ryanmcdermott/clean-code-javascript) - Clean Code concepts adapted for JavaScript
- [Design Patterns for Humans](https://github.com/kamranahmedse/design-patterns-for-humans) - Ultra-simplified explanation of design patterns
- [Refactoring Guru](https://refactoring.guru/) - Design patterns and refactoring techniques

## Algorithms

- [The Algorithm Design Manual](http://www.algorist.com/) - Steven Skiena's Algorithm Design Manual
- [Competitive Programmer's Handbook](https://cses.fi/book/book.pdf) - Free book on competitive programming by Antti Laaksonen

## Web Development

- [You Don't Know JS (book series)](https://github.com/getify/You-Dont-Know-JS) - Deep dive into JavaScript
- [Eloquent JavaScript](https://eloquentjavascript.net/) - Modern introduction to programming with JavaScript
- [The Odin Project](https://www.theodinproject.com/) - Free full-stack web development curriculum
