# Courses and Tutorials

A collection of free courses and tutorials for computer science and software engineering students.

## Full Curricula & Learning Platforms

- [freeCodeCamp](https://github.com/freeCodeCamp/freeCodeCamp) - Full-stack web development and machine learning curriculum with certifications
- [OSSU Computer Science](https://github.com/ossu/computer-science) - Path to a free self-taught education in Computer Science
- [The Odin Project](https://www.theodinproject.com/) - Free full-stack curriculum using Ruby on Rails and JavaScript
- [Khan Academy Computing](https://www.khanacademy.org/computing) - Free computing courses from basics to advanced topics

## Web Development

- [Web Dev for Beginners by Microsoft](https://github.com/microsoft/Web-Dev-For-Beginners) - 24 lessons on JavaScript, CSS, and HTML
- [Full Stack Open](https://fullstackopen.com/en/) - Deep dive into modern web development by University of Helsinki
- [CSS-Tricks](https://css-tricks.com/) - Tips, tricks, and techniques on using CSS

## Data Science & Machine Learning

- [Machine Learning for Beginners by Microsoft](https://github.com/microsoft/ML-For-Beginners) - 26 lessons on classical machine learning
- [AI for Beginners by Microsoft](https://github.com/microsoft/AI-For-Beginners) - 24 lessons on artificial intelligence
- [Data Science for Beginners by Microsoft](https://github.com/microsoft/Data-Science-For-Beginners) - 20 lessons on data science
- [Fast.ai](https://www.fast.ai/) - Making deep learning accessible to all
- [Andrew Ng's Machine Learning on Coursera](https://www.coursera.org/learn/machine-learning) - Foundational ML course

## Programming Languages

- [Exercism](https://exercism.org/) - Code practice and mentorship in 70+ programming languages
- [Rust by Example](https://doc.rust-lang.org/rust-by-example/) - Learn Rust through annotated example programs
- [Go by Example](https://gobyexample.com/) - Hands-on introduction to Go
- [Learn Python the Hard Way](https://learnpythonthehardway.org/) - Structured approach to learning Python

## DevOps & Cloud

- [90 Days of DevOps](https://github.com/MichaelCade/90DaysOfDevOps) - Documenting the learning journey of DevOps
- [AWS Training and Certification](https://aws.amazon.com/training/) - Free digital training from AWS
- [Kubernetes the Hard Way](https://github.com/kelseyhightower/kubernetes-the-hard-way) - Bootstrap Kubernetes from scratch

## Computer Science Theory

- [MIT OpenCourseWare - Computer Science](https://ocw.mit.edu/courses/electrical-engineering-and-computer-science/) - Free lecture notes, exams, and videos from MIT
- [Stanford CS Courses](https://online.stanford.edu/free-courses) - Free courses from Stanford University
- [Nand2Tetris](https://www.nand2tetris.org/) - Build a modern computer system from the ground up
