## Courses & Learning Platforms

- [Elements of AI](https://www.elementsofai.com/) - Free introductory AI course by University of Helsinki and MinnaLearn
- [Google Machine Learning Crash Course](https://developers.google.com/machine-learning/crash-course) - Fast-paced, practical introduction to ML by Google
- [Stanford CS229 Machine Learning](https://cs229.stanford.edu/) - Free lecture notes, problem sets, and materials from Stanford's ML course
- [Stanford CS231n: CNNs for Visual Recognition](https://cs231n.stanford.edu/) - Free lecture notes and assignments on deep learning for computer vision
- [MIT 6.S191 Introduction to Deep Learning](http://introtodeeplearning.com/) - MIT's introductory deep learning course with labs
- [Hugging Face NLP Course](https://huggingface.co/learn/nlp-course) - Free course on NLP using transformers and the Hugging Face ecosystem
- [Kaggle Learn](https://www.kaggle.com/learn) - Free micro-courses on ML, NLP, computer vision, and data science
- [Practical Deep Learning for Coders](https://course.fast.ai/) - Free hands-on deep learning course by fast.ai
- [Deep Learning Specialization](https://www.deeplearning.ai/courses/deep-learning-specialization/) - Five-course specialization by Andrew Ng (audit for free on Coursera)

## AI Tools & Platforms (Free for Students)

- [Google Colab](https://colab.research.google.com/) - Free Jupyter notebooks in the cloud with GPU/TPU access
- [Kaggle](https://www.kaggle.com/) - Free datasets, competitions, and GPU-powered notebooks
- [Hugging Face](https://huggingface.co/) - Open platform for ML models, datasets, and Spaces
- [Weights & Biases](https://wandb.ai/) - ML experiment tracking and visualization, free for students
- [GitHub Copilot](https://github.com/features/copilot) - AI pair programmer, free for students via GitHub Education

## Open Source Frameworks & Libraries

- [TensorFlow](https://www.tensorflow.org/) - Open-source ML framework by Google
- [PyTorch](https://pytorch.org/) - Open-source deep learning framework by Meta
- [scikit-learn](https://scikit-learn.org/) - Free machine learning library for Python
- [LangChain](https://github.com/langchain-ai/langchain) - Framework for building applications with large language models
- [Ollama](https://github.com/ollama/ollama) - Run large language models locally on your machine
- [LlamaIndex](https://github.com/run-llama/llama_index) - Data framework for connecting LLMs to external data
